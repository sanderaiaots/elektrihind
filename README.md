# Tunnitarbimise faili järgi kaalutud keskmine elektirhind

Faili peaks alla laadima elektrilevi.ee lehels. Kahjuks selleks puudub hetkel API.
Alla tuleb laadida CSV fail. (6 või 12 kuud)
